Home

This is a full-stack web application that allows users to manage tasks through a clean, Bootstrap-styled interface. It's structured with a frontend (running on port 5001) that communicates with a separate backend API (running on port 5000). The application implements full CRUD (Create, Read, Update, Delete) operations for tasks, where each task has a name, summary, description, and completion status. The separation of frontend and backend makes it scalable and maintainable, allowing for future expansion or modification of either component independently.

Potential Uses:

Personal Task Management
  • Track personal to-do items
  • Manage daily, weekly, or long-term projects
  • Keep notes and descriptions for complex tasks
Team Project Management
  • Coordinate team activities
  • Track project milestones
  • Share task details and progress
Educational Environment
  • Track student assignments
  • Manage course projects
  • Organize teaching materials
Small Business Operations
  • Track customer orders
  • Manage inventory tasks
  • Schedule maintenance activities

Key Features:

  1. Create new tasks with detailed descriptions
  2. View a list of all tasks in a organized table
  3. Edit existing task details
  4. Delete single or multiple tasks
  5. Responsive design that works on both desktop and mobile
  6. Clean user interface using Bootstrap
